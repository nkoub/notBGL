template<typename graph_t >
void notBGL::save_weighted_edgelist ( std::string  filename,
graph_t &  graph,
vertex_identifier_t  vertexID = vertexID_name,
unsigned int  width = 10 
)

This function saves the weighted edgelist into a file. It is a wrapper of the more general function save_edges_properties(). See the save_properties.cpp example below for further details.

Parameters
filenameName of the file in which the weighted edgelist is to be written.
graphThe graph object.
vertexID[optional] Identifier used for the edges. Options are
  • notBGL::vertexID_name (default): prints the name of the vertices;
  • notBGL::vertexID_num: identifies the vertices with a contiguous sequence of integers starting at 0;
  • notBGL::vertexID_none: properties are printed without identifying the edges.
width[optional] Width of the columns in the file. Default is 10.
